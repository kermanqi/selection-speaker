# 划词朗读器

一个很轻量的 macOS 菜单栏工具：选中英文单词或短句，松开鼠标后自动用 macOS 自带语音朗读。

## 运行

开发运行：

```sh
swift run
```

打包成可双击打开的 App：

```sh
./scripts/build-app.sh
```

生成后的 App 在：

```text
build/划词朗读器.app
```

## 第一次使用

第一次打开后，macOS 会要求授予“辅助功能”权限。请在：

```text
系统设置 -> 隐私与安全性 -> 辅助功能
```

允许“划词朗读器”。授权后重新打开 App。

## 使用方式

1. 打开 App，菜单栏会出现一个扬声器图标。
2. 在网页、PDF、文档或其他应用里划选英文单词。
3. 松开鼠标，它会立即朗读选中的内容。
4. 点菜单栏图标可以暂停自动朗读、停止当前朗读或退出。

目前它会优先朗读含有英文字母的选区，并忽略空白或纯中文选区。若当前应用不直接暴露选中文本，它会在拖选或双击后临时读取剪贴板内容，并立即恢复原剪贴板。
