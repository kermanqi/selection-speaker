# 划词朗读器

一个很轻量的 macOS 菜单栏工具：选中英文单词或短句，松开鼠标后自动用 macOS 自带语音朗读，也可以在鼠标附近显示中文翻译。

## 运行

开发运行：

```sh
swift run
```

打包成可双击打开的 App：

```sh
./scripts/build-app.sh
```

生成后的 App 在：

```text
build/划词朗读器.app
```

## 第一次使用

第一次打开后，macOS 会要求授予“辅助功能”权限。请在：

```text
系统设置 -> 隐私与安全性 -> 辅助功能
```

允许“划词朗读器”。授权后重新打开 App。

## 使用方式

1. 打开 App，菜单栏会出现一个扬声器图标。
2. 在网页、PDF、文档或其他应用里划选英文单词。
3. 松开鼠标，它会立即朗读选中的内容。
4. 点菜单栏图标可以暂停自动朗读、开启中文翻译、设置 DeepSeek API Key、设置模型名、停止当前朗读或退出。

目前它会优先朗读含有英文字母的选区，并忽略空白或纯中文选区。若当前应用不直接暴露选中文本，它会在拖选或双击后临时读取剪贴板内容，并立即恢复原剪贴板。

## 中文翻译

中文翻译默认关闭。开启方式：

1. 点菜单栏图标，选择“显示中文翻译”。
2. 第一次开启时输入 DeepSeek API Key，它会保存到 macOS Keychain。
3. 之后划选英文时，App 会继续朗读英文，并在鼠标附近显示中文翻译。

也可以用全局快捷键快速开关：

1. 点菜单栏图标，打开“偏好设置...”。
2. 在快捷键页面点击“录制快捷键”。
3. 直接按下你想用的组合键。

默认模型是 `deepseek-v4-flash`，可在菜单里修改。

开启中文翻译后，划选文本会发送到 DeepSeek API。单词和短语会显示学习型释义；句子和段落会显示更自然的中文译文。

## 自定义提示词

打开菜单栏里的“偏好设置...”，可以编辑翻译提示词：

- 系统提示词：控制翻译风格和输出格式。
- 用户提示词模板：控制划选文本如何发送给模型，必须保留 `{selectedText}`。

点“恢复默认提示词”可以回到内置默认版本，文本框里的默认提示词可以直接复制走。
